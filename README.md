haapa
=====

A simple status program.

dependencies
------------

required:
* libevent
* cmake

optional:
* libmpdclient
* alsa
* iwlib
* Xlib

building
--------

	cd build
    cmake ..
    make

configuration
-------------

see ```docs/CONFIGURATION.md``` and [this](https://JuhaniImberg.github.io/haapa/)

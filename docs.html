Formats

format_plain

Raw output without colors.

format_i3

Formatted into a streaming JSON array. Suitable for i3bar.

format_i3_manual

Formatted into a streaming JSON array. Suitable for i3bar. Segments are managed manually. You create a new segment with the macro i3_manual_start() and end them with i3_manual_end(color).

format_dzen

Format suitable for use with dzen2.

format_xmobar

Format suitable for use with xmobar.

format_x256

Same as format_plain but with xterm-256color color escape codes. Useful for use with DWM.

format_html

Formatted as HTML with colors. Useful for generating small HTML status strips.

Outputs

string

If the generator returns a string it prints that. If not, it will print the numeric value the generator returned.

percent

Displays the percent of value filling max (value/max)*100 + "%"

bar

Displays a either vertical or horizontal bar of the percent.

timeconv

Converts value into time hh:mm:ss or mm:ss

sizeconv

Converts value into decimal bytes (KB, MB...) with the precision of 2 decimals.

sizeconvi

Converts value into binary bytes (KiB, MiB...) with the precision of 2 decimals.
